## Setup development environment for Python

Golem's Python toolchain is using [uv](https://docs.astral.sh/uv/) as the underlying build tool.
Install it with the following command:

<MultiPlatformCommand
  commands={[
    {
      label: "macOS/Linux",
      command:
        'curl -LsSf https://astral.sh/uv/0.6.17/install.sh | sh',
    },
    {
      label: "Windows",
      command:
        'powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/0.6.17/install.ps1 | iex"',
    },
  ]}
/>
