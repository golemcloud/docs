# Breaking changes June 2024

Several breaking changes are being applied as a preparation of the 1.0 version.

We updated the tooling versions required to build Golem components. The new versions are:

- `wasm-tools@1.210.0`
- `wit-bindgen-cli@0.26.0`
- `cargo-component@0.13.2`

See more details in the [Building Components](/docs/building-components) section.
