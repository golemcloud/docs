# Setting up the Golem TypeScript SDK

<Callout type="warning">
  If the project was created with `golem new`, it already has the Golem TypeScript SDK set up
  and these steps can be ignored.
</Callout>

To add the [Golem JavaScript SDK](https://github.com/golemcloud/golem-ts) to a project, just install it using npm;
<MultiPlatformCommand
  commands={[
    { label: "npm", command: `npm install --save-dev @golemcloud/golem-ts` },
    { label: "pnpm", command: `pnpm add --save-dev @golemcloud/golem-ts` },
    { label: "yarn", command: `yarn add --dev @golemcloud/golem-ts` },
    { label: "bun", command: `bun add --dev @golemcloud/golem-ts` },
  ]}
/>
