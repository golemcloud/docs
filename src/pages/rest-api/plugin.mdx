---|---|---

401|Unauthorized request|`{"error":"string"}`
403|Forbidden Request|`{"error":"string"}`
404|Entity not found|`{"error":"string"}`
409||`{"error":"string"}`
422|Limits of the plan exceeded|`{"error":"string"}`
500|Internal server error|`{"error":"string"}`
