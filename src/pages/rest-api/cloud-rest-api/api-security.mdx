# Api Security API

## Get a security scheme

| Path                                                         | Method | Protected |
| ------------------------------------------------------------ | ------ | --------- |
| `/v1/api/security/{project_id}/{security_scheme_identifier}` | GET    | Yes       |

Get a security scheme by name

**Example Response JSON**

```json copy
{
  "providerType": "Google",
  "schemeIdentifier": "string",
  "clientId": "string",
  "clientSecret": "string",
  "redirectUrl": "string",
  "scopes": ["string"]
}
```

## Create a security scheme

| Path                            | Method | Protected |
| ------------------------------- | ------ | --------- |
| `/v1/api/security/{project_id}` | POST   | Yes       |

**Example Request JSON**

```json copy
{
  "providerType": "Google",
  "schemeIdentifier": "string",
  "clientId": "string",
  "clientSecret": "string",
  "redirectUrl": "string",
  "scopes": ["string"]
}
```

**Example Response JSON**

```json copy
{
  "providerType": "Google",
  "schemeIdentifier": "string",
  "clientId": "string",
  "clientSecret": "string",
  "redirectUrl": "string",
  "scopes": ["string"]

```
