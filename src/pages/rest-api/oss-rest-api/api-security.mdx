# Api Security API

## Get a security scheme
Path|Method|Protected
---|---|---
`/v1/api/security/{security_scheme_identifier}`|GET|No

Get a security scheme by name





**Example Response JSON**

```json copy
{
  "providerType": "Google",
  "schemeIdentifier": "string",
  "clientId": "string",
  "clientSecret": "string",
  "redirectUrl": "string",
  "scopes": [
    "string"
  ]
}
```
